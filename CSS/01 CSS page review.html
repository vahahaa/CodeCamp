For example CSS syntax

Selector:
- Element Selector เช่น h1, h2, ..., h6, p
- Id Selector ใช้สำหรับเลือก element แบบเจาะจง เช่น #para1, #head1
- Class Selector ใช้สำหรับเลือก element ที่ถูกกำหนดอยู่ใน class ต่างๆ เช่น .intro, .info
- Universal Selector ใช้สำหรับเลือก HTML element ทั้งหมดใน page

Property: เป็นคุณสมบัติหรือลักษณะของ element เช่น color, background-color, background-image, border

Value: เป็นค่าของคุณสมบัติหรือลักษณะของ element เช่น blue, black, url, 20px


For example CSS writing

Inline:

<p style="color:red;">example paragraph 1</p>

Internal:

<style> p { color: red; } </style> <p>example paragraph 2</p>

External:

<link rel="stylesheet" href="redtext.css"> <p>example paragraph 3</p>
