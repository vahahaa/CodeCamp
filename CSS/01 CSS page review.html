For example CSS syntax

Selector:
- Element Selector เช่น h1, h2, ..., h6, p
- Id Selector ใช้สำหรับเลือก element แบบเจาะจง เช่น #para1, #head1
- Class Selector ใช้สำหรับเลือก element ที่ถูกกำหนดอยู่ใน class ต่างๆ เช่น .intro, .info
- Universal Selector ใช้สำหรับเลือก HTML element ทั้งหมดใน page

Property: เป็นคุณสมบัติหรือลักษณะของ element เช่น color, background-color, background-image, border

Value: เป็นค่าของคุณสมบัติหรือลักษณะของ element เช่น blue, black, url, 20px


For example CSS writing

Inline:

<p style="color:red;">example paragraph 1</p>

Internal:

<style> p { color: red; } </style> <p>example paragraph 2</p>

External:

<link rel="stylesheet" href="redtext.css"> <p>example paragraph 3</p>

Selector example case

Selector Example Description
#id #firstname Select the element with id="firstname"
.class .intro Select all elements with class="intro"
element.class p.intro Select only p elements with class="intro"
* * Select all elements
element p Select all p elements
element, element,.. div, p Select all div elements and all p elements

Combinator example

Combinator Selector Example Description
element element div p Select all p element inside div element
element > element div > p Select all p element where the parents is a div element
element + element div + p Select the first p element that are placed immediately after div element
element1 ~ element2 p ~ ul Select every ul element that are proceeded by a p element
